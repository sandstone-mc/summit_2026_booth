# The Sandstone booth pack for Smithed Summit 2026

To build the packs, run:
```bash
bun dev:build
// or
sand build
```

To automatically rebuild the packs on each change, run:
```bash
bun dev:watch
// or
sand watch
```

To deploy the pack to our internal build server, run:
```bash
bun deploy
```

Generated pack is in `generated` branch.
